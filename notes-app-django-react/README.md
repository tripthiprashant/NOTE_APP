# Django React Notes App
🔗 **Repository:** https://github.com/tripthiprashant/NOTE_APP

The backend is powered by Django REST Framework, exposing RESTful APIs for efficient data management, while the React frontend delivers a fast and interactive user experience. Tailwind CSS is used to create a clean and modern design with support for both **Dark Mode** and **Light Mode**.

### Features

* Create, edit, and delete notes
* Responsive and modern UI
* Dark and Light mode support
* RESTful API architecture
* Full CRUD operations
* SQLite database integration
* Clean and maintainable code structure

### Tech Stack

**Backend:** Django, Django REST Framework, SQLite

**Frontend:** React.js, Tailwind CSS, Axios

### What I Learned

* Building REST APIs with Django REST Framework
* Integrating React with backend APIs
* Managing application state and data flow
* Implementing CRUD functionality
* Designing responsive user interfaces with Tailwind CSS
* Structuring full-stack web applications

This project demonstrates the integration of a React frontend with a Django backend, showcasing modern full-stack development practices and scalable application architecture.
